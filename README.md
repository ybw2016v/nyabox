# nyabox

一个基于flask、sqlite、redis、jquery、bootstrap4的提问箱。

## 特点

* ~~单页~~三页应用，手搓pjax
* 前后端分离，兼顾og标签
* 使用类Oauth2.0认证，能且仅能通过[DMI](https://m.dogcraft.top)账号登录，登录插件[mkauth](https://github.com/ybw2016v/mkauth)
